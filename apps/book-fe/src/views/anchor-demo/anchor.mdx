## API

### AnchorProps

| 属性       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| container   | 滚动容器           | `Element`         | `window`  |
| offsetX      | 水平偏移量         | `number`         | `0`  |
| offsetY    | 垂直偏移量         | `number`         | `0`  |
| items    | 锚点列表         | `AnchorItem[]`         | `[]`  |

### AnchorItem

| 参数       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| labelledby   | 锚点标签           | `string`         | -  |
| content      | 锚点内容         | `React.ReactNode`         | -  |
| handler    | 锚点点击事件回调         | `React.MouseEventHandler<HTMLLIElement>`         | -  |
| children    | 子锚点列表         | `AnchorItem[]`         | -  |
