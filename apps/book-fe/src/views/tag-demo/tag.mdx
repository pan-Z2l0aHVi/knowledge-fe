## API

### Tag Props

| 属性       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| size    | 大小           | `'small' \| 'medium' \| 'large'`         | `'medium'`  |
| round    | 是否圆角           | `boolean`         | `false`  |
| bordered    | 是否有边框           | `boolean`         | `false`  |
| closable    | 是否展示关闭按钮           | `boolean`         | `false`  |
| color    | 颜色           | `string`         | -  |
| bgColor    | 背景色           | `string`         | -  |
| onClose    | 关闭回调函数           | `() => void \| Promise<void>`         | -  |
