# useDebounce

用来处理防抖函数

## 代码演示

import { useDebounce } from '@youknown/react-hook/src'
import { Button } from '@youknown/react-ui/src'
import { useState } from 'react'

---

### 基本用法

export function BasicUsage() {
  const [value, setValue] = useState(0)
  const run = useDebounce(() => {
    setValue(value + 1)
  }, 500)
  return (
    <div>
      <p className="mt-16px"> Clicked count: {value} </p>
      <Button onClick={run}>Click fast!</Button>
    </div>
  )
}

<BasicUsage />

```tsx
import { useDebounce } from '@youknown/react-hook'
import { Button } from '@youknown/react-ui'
import { useState } from 'react'

export default () => {
  const [value, setValue] = useState(0)
  const run = useDebounce(() => {
    setValue(value + 1)
  }, 500)
  return (
    <div>
      <p className="mt-16px"> Clicked count: {value} </p>
      <Button onClick={run}>Click fast!</Button>
    </div>
  )
}
```

## API

### Type

`(() => void, number) => Result`

### Params

| 参数     | 说明               | 类型         | 默认值 |
| -------- | ------------------ | ------------ | ------ |
| fn       | 需要防抖执行的函数 | `() => void` | -      |
| duration | 防抖时间           | `number`     | `1000` |

### Result

| 参数   | 说明                               | 类型         |
| ------ | ---------------------------------- | ------------ |
| run    | 触发执行 fn，函数参数将会传递给 fn | `() => void` |
| cancel | 取消当前防抖                       | `() => void` |
| flush  | 立即调用当前防抖函数               | `() => void` |
