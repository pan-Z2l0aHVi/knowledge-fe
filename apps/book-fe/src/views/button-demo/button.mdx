## API

| 属性       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| primary    | 主要按钮           | `boolean`         | `false`  |
| disabled   | 禁用状态           | `boolean`         | `false`  |
| size       | 大小               |  `'small' \| 'medium' \| 'large'`  | `'medium'` |
| danger     | 危险按钮           | `boolean`         | `false`  |
| text       | 文本按钮           | `boolean`         | `false`  |
| round      | 大圆角形状         | `boolean`         | `false`  |
| square     | 正方形             | `boolean`         | `false`  |
| circle     | 圆形               | `boolean`         | `false`  |
| loading    | 加载中状态         | `boolean`         | `false`  |
| prefixIcon | 按钮文本前面的图标 | `React.ReactNode` | `null`   |
| suffixIcon | 按钮文本后面的图标 | `React.ReactNode` | `null`   |
