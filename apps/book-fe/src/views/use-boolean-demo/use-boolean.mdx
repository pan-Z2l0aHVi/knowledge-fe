# useBoolean

## 代码演示

import { useBoolean } from '@youknown/react-hook/src'
import { Button } from '@youknown/react-ui/src'

---

### 基本用法

便捷切换 boolean。

export function BasicUsage() {
  const [bol, { setReverse, setTrue, setFalse }] = useBoolean(true)
  return (
    <div>
      <p>Effects：{JSON.stringify(bol)}</p>
      <p>
        <Button onClick={setReverse}>
          Toggle
        </Button>
        <Button onClick={setFalse} className="m-[0_16px]">
          Set false
        </Button>
        <Button onClick={setTrue}>
          Set true
        </Button>
      </p>
    </div>
  )
}

<BasicUsage />

## API

### Type

`(defaultValue: boolean) => [boolean, Action]`

### Params

| 参数       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| defaultValue    | 默认值           | `boolean`         | `false`  |

### Result

| 参数       | 说明               | 类型              |
| ---------- | ------------------ | ----------------- |
| state    | 状态值           | `boolean`         |
| action    | 操作集合           | `Action`         |

### Action

| 参数       | 说明               | 类型              |
| ---------- | ------------------ | ----------------- |
| setTrue    | 设置state为true           | `() => void`         |
| setFalse    | 设置state为false           | `() => void`         |
| setReverse    | state取反           | `() => void`         |
| setBool    | 设置state           | `React.Dispatch<React.SetStateAction<boolean>>`         |
