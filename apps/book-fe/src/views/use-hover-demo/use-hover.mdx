# useHover

## 代码演示

import { useHover } from '@youknown/react-hook/src'
import { Button } from '@youknown/react-ui/src'
import { useEffect } from 'react'

监听 DOM 元素是否有鼠标悬停

---

### 基本用法

export function BasicUsage() {
  const [ele, isHover] = useHover(isHovering => isHovering
    ? <Button>hovering</Button>
    : <Button>text</Button>
  )
  useEffect(() => {
    console.log('isHover', isHover)
  }, [isHover])
  return ele
}

```tsx
import { useHover } from '@youknown/react-hook'
import { Button } from '@youknown/react-ui'
import { useEffect } from 'react'

export default () => {
  const [ele, isHover] = useHover(isHovering => isHovering
    ? <Button>hovering</Button>
    : <Button>text</Button>
  )
  useEffect(() => {
    console.log('isHover', isHover)
  }, [isHover])
  return ele
}
```

## API

### Type

`(boolean => React.ReactNode) => [React.ReactNode, boolean]`

### Params

| 参数       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| render    | 渲染函数           | `boolean => React.ReactNode`         | -  |

### Result

| 参数       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| ele    | 渲染后的元素           | `React.ReactNode`         | -  |
| isHover    | 是否悬停           | `boolean`         | `false`  |
