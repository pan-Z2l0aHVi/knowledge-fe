# useCreation

## 代码演示

import { useCreation } from '@youknown/react-hook/src'
import { Button } from '@youknown/react-ui/src'
import { useState } from 'react'

---

useCreation 的值确保一定不会被重新计算

### 基本用法

export class Foo {
  constructor() {
    this.data = Math.random()
  }
}

export function BasicUsage() {
  const foo = useCreation(() => new Foo())
  const [, setFlag] = useState({})
  return (
    <>
      <p>{foo.data}</p>
      <Button
        onClick={() => {
          setFlag({})
        }}
      >
        Rerender
      </Button>
    </>
  )
}

## API

### Type

`(factory: () => any) => any`

### Params

| 参数       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| factory    | 取值函数           | `() => any`         | -  |
