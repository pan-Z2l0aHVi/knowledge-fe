## API

### Select Props

| 属性         | 说明               | 类型                                | 默认值                             |
| ------------ | ------------------ | ----------------------------------- | ---------------------------------- |
| multiple     | 是否多选           | `boolean`                           | `false`                            |
| disabled     | 是否禁用           | `boolean`                           | `false`                            |
| placement    | 排布               | `Placement`                         | `'bottom-start'`                   |
| filter       | 可筛选             | `boolean`                           | `false`                            |
| placeholder  | 占位符             | `string`                            | `t('react_ui.placeholder.select')` |
| allowClear   | 展示清空按钮       | `boolean`                           | `false`                            |
| noMore       | 没有更多数据       | `boolean`                           | `true`                             |
| menuList     | 可选列表           | `Option[]`                          | `[]`                               |
| value        | 值                 | `string \| number`                  | -                                  |
| defaultValue | 默认值             | `string \| number`                  | -                                  |
| onChange     | 值改变事件回调函数 | `(value: string \| number) => void` | -                                  |
| onLoad       | 加载更多回调函数   | `() => void`                        | -                                  |

### Option

| 参数     | 说明     | 类型                                         | 默认值 |
| -------- | -------- | -------------------------------------------- | ------ |
| label    | 选项 key | `React.ReactNode`                            | -      |
| value    | 选项值   | `(string \| number) \| (string \| number)[]` | -      |
| disabled | 是否禁用 | `boolean`                                    | -      |
