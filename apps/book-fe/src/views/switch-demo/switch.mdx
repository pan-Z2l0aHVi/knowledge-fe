### 受控组件

export function Controllable() {
  const [checked, setChecked] = useState(false)
  return (
    <>
      <Switch value={checked} onChange={setChecked} />
      <Button onClick={() => setChecked(p => !p)}>Toggle</Button>
    </>
  )
}

<Controllable />

## API

### Switch Props

| 属性       | 说明               | 类型              | 默认值   |
| ---------- | ------------------ | ----------------- | -------- |
| size    | 大小           | `'small' \| 'medium' \| 'large'`         | `'medium'`  |
| disabled    | 是否禁用           | `boolean`         | `false`  |
| defaultValue    | 默认值           | `boolean`         | `false`  |
| value    | 值           | `boolean`         | -  |
| onChange    | 值改变回调函数           | `(value: boolean) => void`         | -  |
